### b.访问明细：

栏目里面的内容可以查看用来是用什么方式来到网站的，是用搜索引擎还是直接访问，为了数据的准备性，需要在 CNZZ 屏蔽自己的IP  
cnzz 访问明细里面，可以查看用户具体访问的页面明细，需要点开一下，查看页面是否能正常打开，根据用户访问新闻，确定用户的来源关键词

> 总结：查看网站访问的明细可以找出一规律的内容，目前已知的有：一个相同的IP 会有规律的访问 新闻列表，那肯定来偷文章的

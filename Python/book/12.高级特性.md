# 高级特性

掌握了 Python 数据类型、语句和函数，就能写出许多有用的程序了。

比如构造一个 1-99 的列表，可以通过循环实现：
````python
L = []
n = 1
while n <= 99:
    L.append(n)
    n = n + 2
````

取 list 前一半元素也可以用 循环来实现.

在 Python 中，代码不是越多越少，而是越少越少。代码不是越复杂越好，而是越简单越好。

基于这一思想，我们来介绍Python中非常有用的高级特性，1行代码能实现的功能，决不写5行代码。请始终牢记，代码越少，开发效率越高。
